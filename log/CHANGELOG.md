# Changelog

## v1.1

- Aggiunta coda locale per QSO non sincronizzati tramite `localStorage`.
- Salvataggio QSO via `fetch()` con risposta JSON.
- Visualizzazione in tabella dei QSO non sincronizzati con badge dedicato.
- Sincronizzazione automatica al ritorno online e pulsante manuale "Sincronizza ora".
- Aggiunto totale QSO nella dashboard del log.
- Aggiunti campi QSO completi direttamente nella maschera di inserimento: nome, QTH, locator, note.
- Aggiunta modifica dettagli log tramite `log_edit.php`.

## v1.0

- Login utenti con superadmin.
- Creazione utenti senza registrazione autonoma.
- Log multipli per utente.
- Inserimento rapido QSO.
- Export ADIF per singolo log.
- UI Bootstrap 5 responsive.
